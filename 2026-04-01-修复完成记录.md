**修复：**
- 删除了所有项目页面中的 `<script setup>` 代码块

**受影响文件（14 个）：**
- `projects/*.md` - 6 个英文项目页
- `zh/projects/*.md` - 8 个中文项目页

### 2. 移除 GitHub Stars 显示

**问题：** 硬编码的 Stars/Downloads 值且使用不兼容语法

**修复：**
- 删除了所有项目页面中的 GitHub Stars 行
- 删除了所有项目页面中的 Downloads 行（中文页）
- 更新了 `template.md` 模板
- 更新了 `README.md` 数据源说明

### 3. 精简主页链接

**修复：**

- 保留 ClawHub 和 GitHub 两个主要链接

**受影响文件：**
- `index.md`

---

### 4. 增强 CSS 强制暗色

**修复：**
- 在 `custom.css` 顶部添加 HTML/Body 强制暗色规则
